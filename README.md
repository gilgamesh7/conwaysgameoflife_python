# 1. What the project does
## conwaysgameoflife_python
- Implement Conway’s Game of Life algorithm with Python 
- Build a curses view to display the Game of Life grid 
- Create an argparse command-line interface for the game 
- Set up the game app for installation and execution

1.  __init__.py enables rplife/ as a Python package.
1. __main__.py works as an entry-point script for the game.
1. cli.py contains the command-line interface for the game.
1. grid.py provides the life grid implementation.
1. patterns.py and patterns.toml handle the game’s patterns.
1. views.py implements a way to display the life grid and its evolution.
